    StoryAI is an innovative application designed to craft unique and engaging stories tailored for children across various age groups. Leveraging the power of generative AI. The application lets you configurate and generate unique stories with help of AI.

    React Native
    OpenAI
    Node.js
    Firebase
    Redux

    This project was most focused on OpenAI and how to formulate prompts to recieve consistently formatted replies. This project therefore lacks in design but is more heavy on backend aspects.
